# Lawncer

🌱🤖🔥📐🎲

Tactical lawn care with suspiciously familiar mechanics.

**[Play Now](https://chaikabogdan.github.io/Lawncer/)**

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm

### Installation

```bash
pnpm install
```

### Development

Start the development server:

```bash
pnpm dev
```

The app will be available at `http://localhost:5173`

### Building

Build for production:

```bash
pnpm build
```

Preview the production build:

```bash
pnpm preview
```

## Deployment

This project automatically deploys to GitHub Pages when you push to the `main` branch using GitHub Actions.

## Tech Stack

- **Vite** - Fast build tool
- **TypeScript** - Type-safe JavaScript
- **pnpm** - Package manager
